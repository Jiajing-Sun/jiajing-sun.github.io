## Abstract

**中文摘要**：  
本文探讨了计数数据自回归(INAR)过程的模型平均，并提出了基于极大似然估计的K折交叉验证模型平均(KCVMA).INAR过程被广泛应用于医学、经济学、金融和市场营销等领域中非独立的计数数据的建模.与传统模型不同，INAR过程不是通过简单的加乘关系，而是在概率分布层面刻画数据生成过程(DGP).INAR模型一般采用极大似然(ML)法估计，因此KCVMA不仅可以在点、区间上，更可以在整个概率分布层面上对计数数据进行预测.此外，本文提出的KCVMA估计量在平方损失准则下具有渐近最优性，且只要一个候选模型没有被欠拟合，KCVMA估计量就具有一致性.蒙特卡洛模拟以及实证分析的结果表明，与现有的模型选择方法以及模型平均方法相比，KCVMA可以更准确地对计数数据进行预测.最后，本文所提出的方法还可以推广至基于极大似然估计量的任意非线性、非独立数据的模型平均上.

**English Abstract**:  
This article considers model averaging in the class of integer-valued au-toregressive (INAR)processes and proposes a K-fold cross-validation model averaging(KC'VMA) method. The INAR, process is a model class that can be used to modeldependent count data in fields such as medicine, economics, finance, and marketing.It specifies the data generating process (DGP) as a probability distribution instead ofusing approximate multiplicative and additive relationships, which means that maxi-mum likelihood (ML)is a natural candidate for estimation. Forecasts can be evaluatednot only in terms of points and intervals but also of the entire forecast distribution.Additionally, the KC'VMA estimator is asymptotically optimal by a metric equivalentto a squared loss criterion. The KC'VMA estimators are consistent, provided that atleast one candidate model is not under-fitted. Monte Carlo simulations and empir-ical analysis show that the KCVMA estimator can improve forecast accuracy overexisting model averaging and model selection methods. Finally, our technique can begeneralized to the model averaging of any set of non-linear dependent data modelsthat utilize maximum likelihood estimation (MLE).
